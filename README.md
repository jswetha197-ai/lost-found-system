# Lost and Found System (Java OOP Project)

## 📌 Overview

This project is a Java-based Lost and Found Management System developed using Object-Oriented Programming concepts. It helps users report lost items, upload found items, and manage claims efficiently.

## 🚀 Features

* User login system
* Upload lost items
* View found items
* Claim verification process
* Admin dashboard
* Student & security dashboards

## 🛠️ Technologies Used

* Java
* OOP concepts (Classes, Objects, Inheritance, etc.)
* Java Swing (for GUI, if applicable)

## ▶️ How to Run

1. Open terminal in project folder
2. Compile:

   ```
   javac *.java
   ```
3. Run:

   ```
   java Main
   ```

## 📂 Project Structure

* `Main.java` → Entry point
* `LoginScreen.java` → User login
* `AdminDashboard.java` → Admin controls
* `UploadLostItem.java` → Add lost items
* Other classes handle different features

## 📌 Note

This is an academic project created to demonstrate OOP concepts in Java.
